# 元识・星分身 项目规划

## 1. 项目描述

**产品定位：** AI 分身前置社交平台，核心理念「让 AI 先替你遇见，对的人无需试探」

**目标用户：** 社恐人群、追求深度交友、厌倦浅尝辄止社交的年轻用户

**核心价值：** 铸造分身 → AI 匹配 → 分身 24h 对聊 → 生成报告 → 双向确认 → 真人开聊，全程零真人前置社交压力

**设计风格：** 深空宇宙科技 + 神秘暗调，主色深空黑 #0F0F1E、星尘紫 #6C5CE7、极光蓝 #00D1FF、银白微光 #E0EFFF

## 2. 页面路由结构

- `/` - 星核铸造页（创建 AI 分身）
- `/matching` - 星轨匹配页（AI 分身自动寻缘）
- `/dashboard` - 星语看板页（分身 24h 实时对话窗）
- `/report` - 星鉴报告页（24h 聊天最终评估）
- `/unlock` - 星联解锁页（双向确认→真人开聊）

## 3. 核心功能列表

- [x] 星核铸造页：极简人格复刻，3 分钟完成
- [x] 星轨匹配页：AI 分身自动寻缘，全程神秘等待
- [x] 星语看板页：静默观察 AI 对聊，透明但不干预
- [ ] 星鉴报告页：24h 聊天最终评估，数据+感性双维度
- [ ] 星联解锁页：双向确认→真人开聊

## 4. 数据模型设计

（当前阶段使用 Mock 数据，无需数据库）

### 分身数据（本地）
- 星核 ID、人格标签、分身形象、匹配偏好、聊天底线

### 匹配数据（Mock）
- 匹配状态、对方分身信息、24h 倒计时、聊天进度

### 报告数据（Mock）
- 同频总分、三大评估维度、聊天名场面、AI 建议

## 6. 开发阶段计划
